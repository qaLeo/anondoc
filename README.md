# AnonDoc

Инструмент для обезличивания персональных данных в документах. Работает полностью локально в браузере — данные не покидают ваш компьютер.

## Возможности

- **Анонимизация** — автоматически находит и заменяет ПД на токены (`[ФИО_1]`, `[ТЕЛ_1]` и т.д.)
- **Деанонимизация** — восстанавливает оригинальный текст по сохранённой карте замен
- **Поддерживаемые форматы** — TXT, DOCX, XLSX, PDF (текстовые)
- **Обнаруживаемые типы ПД** — ФИО, телефон, email, ИНН, СНИЛС, паспорт, дата рождения

## Как использовать

### Анонимизация
1. Откройте вкладку **Анонимизация**
2. Загрузите документ (перетащите или нажмите на зону загрузки)
3. Нажмите **Анонимизировать**
4. Проверьте результат и при необходимости отредактируйте вручную
5. Сохраните файл или скопируйте текст

### Деанонимизация
1. Откройте вкладку **Деанонимизация**
2. Загрузите ранее анонимизированный TXT-файл
3. Нажмите **Деанонимизировать** — карта замен подтягивается автоматически
4. Сохраните восстановленный документ

## Технические детали

- Карта замен (токен → оригинал) хранится в `localStorage` браузера
- Автоматически удаляется через 30 дней
- Никакие данные не отправляются на сервер

## Стек

- React 19 + TypeScript + Vite
- Tailwind CSS
- mammoth.js (DOCX), SheetJS (XLSX), pdfjs-dist (PDF)
